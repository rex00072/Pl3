Code to find digits in an Integer

Problem Statement: Write a code in any programming language that counts the digits in a number. 


	# Iterative Python program to count 
	# number of digits in a number 
		
	def countDigit(n): 
		count = 0
		while n != 0: 
			n //= 10
			count+= 1
		return count 
		
	# Driver Code     
	n = int(input())
	print ("Number of digits : % d"%(countDigit(n)))

Sample Input

Enter an integer: 6989

Sample Output

Number of digits: 4

Variables used in the program

count, n

Time Complexity

O(log10(n))
